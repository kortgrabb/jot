# Installation

## Prerequisites

- Rust toolchain (rustc, cargo)

## Installing OxidLog

### From crates.io

```bash
cargo install oxidlog
```

### From source

```bash
git clone https://github.com/kortgrabb/oxidlog.git
cd oxidlog
cargo install --path .
```

## Updating OxidLog

```bash
cargo install --force oxidlog
```

## Uninstalling OxidLog

```bash
cargo uninstall oxidlog
```
